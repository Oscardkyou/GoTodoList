# To-Do List на Go

Это простой to-do list API, написанный на Go, который позволяет пользователям управлять своими задачами.

## Функции

- Создание задачи
- Получение списка всех задач
- Обновление описания задачи
- Отметка задачи как выполненной
- Удаление задачи
